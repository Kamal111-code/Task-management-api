# Task Management API (Todo Project)

A simple **Todo application** built using **Spring Boot**.  
This project demonstrates basic CRUD operations and Spring Boot fundamentals.

## Current Features
- Add new tasks
- View tasks
- Update tasks
- Delete tasks

## Tech Stack
- Java 17
- Spring Boot
- Maven

## Planned Enhancements
- Replace H2 with MySQL
- Add DTO layer
- Input validation
- Global exception handling
- Pagination and sorting
- Security (JWT / Role-based access)
- Unit and integration tests

## Getting Started
1. Clone the repo
2. Run the project using `mvn spring-boot:run`
3. Access tasks via provided endpoints (Postman / browser)
